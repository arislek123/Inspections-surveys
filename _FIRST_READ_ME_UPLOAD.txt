IMPORTANT - HOW TO UPLOAD TO GITHUB

1) Right-click this ZIP and choose Extract All.
2) Open the extracted folder.
3) Select ALL items inside this folder, including:
   .github, assets, public, src, package.json, vite.config.ts, etc.
4) Drag/drop ALL selected items to GitHub.
5) Do NOT upload this ZIP file itself.
6) Do NOT upload files one-by-one from inside src/components.

Correct GitHub root must show:
.github/
assets/
public/
src/
index.html
package.json
package-lock.json
vite.config.ts
